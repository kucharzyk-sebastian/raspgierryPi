# Raspgierry Pi
Equivalent of the classic Brick Game recreated in Python for Raspberry Pi 3

## Hardware

* Raspberry Pi 3B+
* Touchscreen Waveshare A - LCD TFT 3.5"
* SNES USB gamepad

## Software

* Operating system: Raspbian 4.14
* Python: 3.53
* PyGame: 1.9.3
